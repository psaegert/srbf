# srbf brand assets

The canonical brand of `srbf` is the results-site header lockup: the lowercase wordmark **srbf**
in JetBrains Mono SemiBold (weight 600, letter-spacing −0.02em), indigo `#4f46e5` on light
backgrounds and periwinkle `#8b8bff` on dark ones, next to the subtitle
**Symbolic Regression Benchmark Framework** (Inter).

| File | Use |
|------|-----|
| `srbf-wordmark.svg` | Wordmark for light backgrounds (`#4f46e5`). Outlined paths, no font dependency. |
| `srbf-wordmark-dark.svg` | Wordmark for dark backgrounds (`#8b8bff`). |
| `srbf-icon.svg` | Rounded-square app tile: white wordmark on indigo. Favicon / docs logo / avatar. |
| `apple-touch-icon.png` | 180×180 PNG render of the tile for iOS home screens. |
| `srbf-social.png` | 1280×640 social-preview card (GitHub repo settings → Social preview; also the site's `og:image`). |

Copies deployed with the results site live in `../../results-site/`; the MkDocs logo/favicon copy
lives in `../../docs/assets/`.

The wordmark SVGs are generated by outlining the glyphs of JetBrains Mono SemiBold
(<https://github.com/JetBrains/JetBrainsMono>, OFL-1.1) with fontTools, applying the −0.02em
letter-spacing of the site's `.brand-mark`; the social card is rendered from the same lockup with
the site's webfonts.
