About these numbers

These series are re-rendered through srbf.analysis from the archived Flash-ANSR v23.0 evaluation (the first paper's results), not yet a fresh canonical run. Flash-ANSR series carry version v23.0; symbolic-data, srbf, and the catalog artifacts are - (they did not exist at eval time). The version travels with each series in the legend.

The compute axis is median per-problem wall-clock fit time, the one budget comparable across methods (Flash-ANSR sweeps choices, PySR iterations, NeSymReS the beam width, E2E the candidate count; time unifies them).

Metrics. The two headline metrics are the strict is_perfect_fit numeric recovery (vNRR: FVU on the clean validation targets at or below float32 epsilon) and the expression length ratio (predicted / ground-truth prefix length; 1 is ideal). The full menu is under More metrics. Rate metrics count a failed prediction as 0 (never conditioned on success); diagnostic metrics (FVU, token overlap, edit distances, lengths) are computed only where a prediction exists. Recovery and FVU always score against the clean targets, including in the noise sweep. Token-overlap and exact-match metrics compare raw prefixes: archived predictions are not re-normalized and the baselines tokenize variables differently, so treat cross-series comparisons of those with care. log-probability and selection score exist only for Flash-ANSR series.

Privacy & cookies

This site is a static page with no analytics, no tracking, and no third-party cookies. It stores exactly one first-party cookie (srbf_colors), and only after you change a series colour: it holds your colour overrides so the chart looks the same when you return on this device. It contains no personal data and is never sent anywhere — the page is fully client-side. Use Reset all colours in the explorer to delete it at any time. Because it is set solely to fulfil a preference you explicitly request and carries no tracking, it is a functional/strictly-necessary cookie under the GDPR / ePrivacy Directive and requires no consent banner; this notice is provided for transparency.
